# IDEA:
Counter:
- Counter.h đóng vai trò route / manager, mọi code ngoài khi dùng đều sẽ đi vào code này
- Init: tạo 2 worker ở theard khác (chỉ được tạo 2 worker duy nhất)
- Close: xóa worker, đóng các theard liên quan, xóa luôn pending nếu có
- Count (phần quan trọng nhất):
+ xét ext của path:
++ nếu là file ảnh -> gọi callback ngay với pages = 1
++ nếu là file pdf -> gọi đẩy vào pending của Pdf worker 
++ nếu là file docx -> gọi đẩy vào pending của docx worker 

PdfWorker:
- từ pending tạo các theard phụ nhỏ hơn chạy song song để lấy pages (có limit số theard phụ là 10 và sẽ được define ở trong PdfWorker.h)
- sau khi 1 theard lấy page xong -> gọi callback

DocxWorker:
- check có word trên máy không (chỉ check 1 lần khi init):
+ nếu không: gọi callback ngay với pages = 0 và error="Missing Word"
+ nếu có:
++ gọi Work com
++ gọi tuần tự các file trong pending để lấy pages, lấy xonog cái nào thì callback cái đó
