++++ xoay bmp gốc 90 độ
+++ if (orientation == "Auto")
++++ nếu w bmp gốc > h bmp gốc -> xoay 90 độ, không thì skip
++ scale lại bmp gốc vào bmp mới (nhớ sử lí để né unprintable):
+++ scale = "No Scale" -> chỉ đưa vào canvas, không scale
+++ scale = "Fit to page (keep aspect ratio)" -> scale nhưng giữ ratio
+++ scale = "Fill page (ignore aspect ratio)" -> fill canvas, không quan tâm ratio
+ đẩy máy in với paper = cfg.paper và orientation = "Portrait" (ép printer in giống 1:1 nhất có thể):
++ tạo Job:
+++ đẩy ảnh vào (đẩy 2 cái nếu PrintDuplex và 1 nếu PrintSimplex)
+++ chờ flag từ printer hoặc spooler (chọn cách cân bằng giữa chính xác, đơn giản):
++++ nếu thành công -> return 0 error = ""
++++ nếu fail -> return 1 v error 



+ win.SetResourceProcessLabel(L"Rendering resources...");

++++ win.SetResourceProcessColor("red");
