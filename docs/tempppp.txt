++ xem pages trả về có error không (nếu có thì error != empty)
+++ nếu error:
++++ win.SetResourceProcessColor("red");
++++ chạy 1 messengebox bằng winapi log lỗi ra và cho user chọn:
+++++ 'skip this page' -> skip + win.SetResourceProcessColor("green") + win.SetResourceProcess(tổng số page còn lại, số page đã render thành công);
+++++ 'cancel' -> sẽ set cancelFlag = true + return

++++ win.SetResourceProcess(tổng số page còn lại, số page đã render thành công);
+ sau khi tất cả cb đã được trả về mà không có cancel giữa chùng thì return
